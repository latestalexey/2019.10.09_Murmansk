	Наименование	Тип	Описание
1.	Результат		Содержит в себе результат выполнения произвольного кода.
2.	СсылкаДокумент1С	ДокументСсылка	Ссылка на выгружаемый (исходный) документ 1С.
3.	ДанныеДляВыгрузки_Шапка	Структура	"Структура, подготовленная на основании правил обмена выгрузки полей шапки документа 1С в шапку документа Mobile SMARTS. Вне зависимости от установленного признака ""Выгружать"" в структуру входят значения, рассчитанные по типам источников:
- Реквизит 1С (все значения из правил обмена);
- Произвольное значение (все значения из правил обмена);
- Произвольный код (только рассчитанные на момент исполнения последующего произвольного кода).
Ключами структуры являются значения, указанные в колонке ""Поле Mobile SMARTS"" (ИмяПриемника) правил обмена."

	Для данного произвольного кода не существует доступных элементов структуры.
